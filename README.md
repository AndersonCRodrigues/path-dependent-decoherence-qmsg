# Q-MSG: Biographical Irreducibility in Quantum Mechanics

**"Whoever seeks the answer loses the question; Whoever guards the enigma finds both."**

This repository contains the complete scientific validation for the Q-MSG thesis, demonstrating that a quantum system's "biography" (its dynamical trajectory, $K$) contains physically real, measurable information that is not captured by its instantaneous state ($\rho$).

We prove mathematically and demonstrate via simulation that two quantum processes (twins) with identical unitaries ($U_A = U_B$) and identical final states ($\rho_A = \rho_B$) will diverge measurably under decoherence ($\rho'_A \neq \rho'_B$) if their underlying biographies ($K_A \neq K_B$) differ.

##  How to Reproduce All Results

You can reproduce all figures and results presented in the paper by running the main script.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/AndersonCRodrigues/qmsg-biographical-irreducibility.git](https://github.com/AndersonCRodrigues/qmsg-biographical-irreducibility.git)
    cd qmsg-biographical-irreducibility
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the main script:**
    ```bash
    python code/qmsg_complete.py
    ```
    This runs all 6 tests in sequence (~30 seconds) and generates all figures in the `/figures` directory.

4.  **Validate the Results:**
    The script will also print the results to the console. You can then compare this output against the original execution log provided in this repository to validate the results:
    `code/run_final.log`
